# Betriebssysteme und Netzwerke 2017 - Pflichtkurs Bachelor Wirtschaftsinformatik

Dieses Repository beinhaltet alle Materialien zum Pflichtkurs "Betriebssysteme und Netzwerke" an der Berufsakademie Schleswig-Holstein.

Für jede Vorlesung ist ein nummerierter Ordner angelegt. Das Material beinhaltet:

* "\_vorlesung" - markdown Datei mit dem Inhalt der Vorlesung
* "\_uebung" - Anleitung für den Übungsteil
* "/uebung/\*" - Dateien für den Übungsteil


## Inhalt

* 27.04 - Einführung Computergeschichte, Aufbau von Computersystemen
* 04.05 - Betriebssyteme: Geschichte, Aufbau, Speicherverwaltung
* 12.05 - Betriebssysteme 2
* 19.05 - Betriebssystempraxis
* 25.05 - Netzwerktheorie / Himmelfahrt 
* 01.06 - IPv4 und IPv6 
* 08.06 - Diverse Netzwerkprotokolle
* 15.06 - Netzwerkpraxis 
* 22.06 - Klausurenvorbereitung
* 29.06 - Klausurenwoche
